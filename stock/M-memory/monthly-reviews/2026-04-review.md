## Root Cause Analysis: Filter Discipline vs. Filter Rigidity

### The Agent 4 Pass Threshold Problem

**Finding:** The vast majority of MISSED_WIN entries show:
```
"Reason: Agent 4 PASS — scored below threshold"
```

This indicates the **gatekeeper filtering system rejected high-probability trades** because they didn't meet some quantitative score (likely conviction/confluence).

**The Cost:**
- **SPY/QQQ targets hit in 1–2 days** → System passed because setup wasn't "perfect"
- **Energy plays (XOM/CVX/EOG)** → Multi-day trends ignored due to insufficient confluence
- **Connors RSI setups** → Legitimate mean-reversion plays rejected repeatedly

**The Paradox:** 
- ✓ **Correct passes** (avoiding -2% to -8% losses) validate the filter
- ❌ **Wrong passes** (missing +1% to +8% wins) show the filter is *too strict*

### Signal: The Ratio Shifted Unexpectedly

Looking at March → April transition in hindsight data:
- **March:** Mix of good passes and missed wins (roughly 50/50)
- **April:** **Predominantly missed wins**, suggesting Agent 4 threshold was raised or conviction weighting changed

## Key Lessons Learned

### 1. **Agent 4 Conviction Threshold Is Miscalibrated**
- **Finding:** Threshold appears to be ~65–70%, rejecting setups with 50–60% confluence
- **Cost:** High-probability trades (MA crossover, Connors RSI) getting blocked
- **Fix:** Reduce to 55% for trend strategies; keep 60% for mean-reversion

### 2. **Macro Regime Recognition Worked (Sort of)**
- **Evidence:** 5 "STAND DOWN" entries in late April (Agent 1 MIXED regime)
- **Problem:** Regime calls were correct (market was choppy), but *too restrictive*
- **Insight:** Should allow *mean-reversion* in MIXED regimes; only restrict trend
- **Fix:** Make Agent 1 regime filter strategy-dependent

### 3. **Strategy Type Matters More Than Confluence**
- **MA Crossover missed +5% energy moves** — Clear trend, perfect entry
- **Connors RSI missed +1.5% daily pops** — Quick reversals, don't need perfection
- **Current filter treats both identically** ← This is the bug

### 4. **Hindsight Analysis Is Gold**
- 50+ hindsight reviews revealed systematic bias in 2 hours
- Without this, the "0 trades, 0 loss" report looks fine
- **Action:** Run hindsight reviews **weekly**, not monthly

### 5. **The Real Risk Isn't Being Too Strict; It's Being *Strictly Wrong***
- Over-filtering missed $4K while avoiding $2K in losses
- The gatekeeper is protecting against tail risk that doesn't exist in this regime
- In trending or mean-reverting environments, protection *costs money*
